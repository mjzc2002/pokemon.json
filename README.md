# Gestor de Equipo Pokémon Competitivo
Aplicación de consola en Python para gestionar tu equipo Pokémon. Permite ver, agregar y buscar Pokémon, guardando todo en un archivo JSON para que persista entre sesiones.
## ¿Qué hace?

Ver los Pokémon de tu PC
Agregar nuevos Pokémon con nombre, habilidad y nivel
Buscar un Pokémon por nombre
Guarda automáticamente los datos en pokemon.json

## Tecnologías

Python 3
Librería json (incluida en Python, no necesitas instalar nada)

## ¿Cómo ejecutarlo?
bashpython pokemon_competitivo.py
## Aprendizajes aplicados

Manejo de archivos JSON (lectura y escritura)
Funciones, diccionarios y listas
Menú interactivo con bucle while
Manejo de errores con try/except
